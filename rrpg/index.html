The Learning Rust Role Playing Game

The Learning Rust RPG

This is an interactive set of tutorials, binded in a structured linear story. It borrows from branching stories, interactive Fiction and gamebooks Its a less physical implementation of a live action role-playing game LARP

Task Difficulty

  1. Routine Anyone can do this basically every time.

  2. Simple Most people can do this most of the time.

  3. Standard Typical task requiring focus

  4. Demanding Requires full attention

  5. Difficult Trained people can do this

  6. Challenging With Training, skills or great effort, it can be done
